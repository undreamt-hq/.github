# Undreamt

> Building what has not yet been built.

Undreamt is an open-source organization focused on creating developer tooling, educational platforms, frameworks, infrastructure, AI systems, and experimental software ecosystems.

We build ambitious projects in public, share what we learn, and explore ideas that prioritize long-term maintainability, clean architecture, and developer experience.

---

## About

Undreamt exists as a home for experimentation, learning, and engineering.

Rather than focusing on a single product, the organization develops a collection of projects that explore different areas of software development, education, artificial intelligence, infrastructure, and developer tooling.

Our goal is to create software that is useful, understandable, extensible, and built to last.

---

## Philosophy

We believe software should be:

* Learnable
* Modular
* Extensible
* Developer-first
* Open by default
* Built in public

We value:

* Clean architecture
* Strong foundations
* Thoughtful engineering
* Long-term maintainability
* Continuous learning
* Open collaboration

---

## Ecosystem

### [Orbit](https://github.com/orbit-projects)

A modular Python ecosystem focused on application architecture, developer tooling, dependency injection, type safety, and scalable software design.

### [Gabby](https://github.com/undreamt-hq/gabby)

Adaptive intelligence infrastructure for building AI systems, agents, memory architectures, orchestration pipelines, and evolving workflows.

### [Learn Undreamt](https://github.com/learn-undreamt)

An educational initiative focused on teaching software engineering, architecture, programming, open source, and modern development practices through structured learning resources.

### [Point](https://github.com/undreamt-hq/point)

Educational authoring language and documentation toolkit for building lessons, courses, learning paths, glossaries, knowledge graphs, and educational websites.

* Developers
* Students
* Researchers

Curiosity, experimentation, and a willingness to learn are enough.

## Vision

Undreamt is being built as a long-term ecosystem of projects, tools, educational resources, and communities.

We want to create software that is:

* Useful
* Educational
* Well engineered
* Open and collaborative
* Built for the long term

Our ambition is not simply to create projects, but to create foundations that help others learn, build, and experiment.

---

## Status

Undreamt is currently in its early stages.

Projects are actively evolving and many ideas remain experimental.

APIs, architectures, and implementations may change as the ecosystem matures.

We build in public, iterate continuously, and learn along the way.
